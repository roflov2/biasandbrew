# Bias and Brew Website

Personal consultancy website for **Konstantin Pokaluhin** — Ethical AI solutions for government.

## Tech Stack

- **React 19** + **Vite**
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Lucide React** for icons

## Getting Started

```bash
cd bias-and-brew
npm install
npm run dev
```

## Deployment

Configured for **Cloudflare Pages**:
- Build command: `npm run build`
- Output directory: `dist`

## Contact Form

Uses [Web3Forms](https://web3forms.com/). Access key is already configured in `Contact.jsx`.
